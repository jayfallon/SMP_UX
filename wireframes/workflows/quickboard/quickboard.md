# Quickboard

On every view of the SMP website there should be a Quickboard button present in the user bar that should be activated open/close by clicking on the button itself. To successfully build and submit a Quickboard, the user would have to at least favorite one image on SMP and activate the submit button on the board. Optional attributes for the board would be Name, Layout, # of images. One a board is successfully submittted, the user will be provided with a short URL for sharing and social media options as well.
