# Membership Success

In order to successfully activate a member account on SMP, a user must first provide a valid email address to which theu have ownership. Once the user has provided a valid email address, SMP will send them an activation email containing a link back to SMP for account activation. When the user activate the link in their email, they will be redirected via browser to an interstitial view on SMP containing an account activation form. The user will need to provide SMP with a valid username and a valid password, confirm their password, accept SMP's ToS and submit their activation form. Should the user-provided information be valid, the account will be activated and the new member will be redirected to an interstial page containing visually enhanced links to other parts of the SMP website.
